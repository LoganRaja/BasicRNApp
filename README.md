# Installation

```
$ git clone https://github.com/LoganRaja/BasicRNApp.git
$ npm install
```

# iOS

```
$ cd ios
$ pod install
```

# iOS or Android debug or .env

```
$ npm run ios
$ npm run and
```

# iOS or Android production or .env.prod

```
$ npm run ios:prod
$ npm run and:prod
```

# iOS or Android staging or .env.staging

```
$ npm run ios:stg
$ npm run and:stg
```
